# Service Network Website

## Overview
This project is the initial site creation for **CSE 340: Server-side Development**.  
It demonstrates server-side fundamentals using **Node.js, Express, and EJS** templates, with deployment to **Render.com**.

## Features
- Node.js + Express server running locally and deployed to Render
- EJS templates for:
  - Home
  - Organizations
  - Projects
  - Categories
- Shared EJS partials for header and footer
- Navigation bar linking all pages
- Static assets served from `public/`:
  - CSS in `public/css/style.css`
  - Images in `public/images/`
- Professional, responsive, and accessible design
- GitHub repository excludes private files (e.g., `.env`)
